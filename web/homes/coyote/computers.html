SpaceBook i9

Apple MacBook Pro 15" (2019)

This is my main mobile workstation - I really like UNIX but never get on well with Linux, so macOS is a good choice for me.

Despite some issues I've had with similar Macs, this does everything I need. It has modern I/O I actually like, and a great display. I don't mind the keyboard too much.

This one's called SpaceBook i9 because I've had a space gray MacBook and a previous 2016 MacBook Pro which was SpaceBook Pro. i9's were pretty new when this came out, so I put it in the name.

Specifications

Model: Apple MacBook Pro, 15-inch, 2019
Processor: Intel Core i9 9880H 8c/16t @ 2.3GHz
Graphics: Intel UHD Graphics 630, AMD Radeon Pro 560X
Memory: 16GB 2400MHz DDR4
Storage: 512GB NVMe SSD
Display: 2880x1800 15.4 inch built in display
Mouse: Logitech MX Anywhere 2S
OS: Apple macOS 11
Shell: zsh
Owned since: November 2019
