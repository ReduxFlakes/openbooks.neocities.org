OpenBooks Search

OpenBooks Search shares a lot with the older search engine, VHSearch, except that it provides a more modern UI and is built into OpenBooks rather than being a separate website.


VHSearch

VHSearch is a search engine that only searches Neocities - if it isn't hosted on Neocities, it doesn't show up in search results. While created by the same people who created OpenBooks, it is not part of OpenBooks itself.
