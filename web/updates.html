OpenBooks 9 Series

9.5.1 - August 4, 2021

  • Core pages now use an entirely new layout.
  • Layout is no longer basedon OpenBooks 8 from 2019.
  • Core pages are now responsive and mobile compatible.

9.5 - August 3, 2021

OpenBooks 9.5 introduces a new home page design.

  • Home page layout now displays logo centrally.
  • Tiles are now larger and more centrally located.
  • Navigation sidebar is now more detailed and text is smaller.
  • Minor design tweaks to layout of home page.
  • Preparations made for design tweaks in next major revision of OpenBooks.
  • Refined shades of blue.
  • Added OpenBooks 9.4 to TimeWarp.

9.4.3 - July 25, 2021

  • Tweaked About page.
  • Tweaked design of TimeWarp page.
  • Relocated WebRing navigation on home page.
  • Page titles are now capitalized site-wide.

9.4.2 - July 17, 2021

  • About page now provides a description of the site as well as its purpose.
  • Updated main page design.
  • Search box added to main page.
  • Fixed performance issue while scrolling in WebKit browsers - background is no longer fixed in position.
  • Soft shadow added to top of pages to prevent hard cut off of site content against the browser chrome in Safari.
  • Rounded the corners of various site design elements.
  • Refined shades of blue.

9.4.1 - July 8, 2021

  • Status bar is no longer attached to the top of the page - this should make it look more seamless with the new Safari.
  • Rounded corners of page elements to also blend in with the new Safari.
  • Links are less broken on TimeWarp page.

9.4 - June 20, 2021

OpenBooks 9.4 includes under the hood improvements for page load speed, and a new design updates.

  • TimeWarp has a new design.
  • Status bar now displays animated load indicator for most pages.
  • CSS separated based on what elements are actually used on a given page - as a result, the home page should load faster.
  • Images optimized on WebRing page.
  • Sitemap uses Lite CSS.
